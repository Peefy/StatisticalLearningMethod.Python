## 统计学习

统计学习就是计算机系统同构运用数据及统计方法提高系统性能的机器学习

统计学习方法三要素:模型,策略和算法

统计学习的对象是数据,它从数据出发,提取数据的特征,抽象出数据的模型,发现数据中的知识,又回到对数据的分析与预测中去.

统计学习的前提:关于数据的基本假设:同类数据具有一定的统计规律性
